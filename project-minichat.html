Mini Chat

Demande du client :

Le client souhaite développer un mini système de messagerie instantanée avec des fonctionnalités de base, destiné à un site ou une application.

  • Client : Start-up web / Projet pédagogique
  • Objectif : Créer un mini-chat simple et fonctionnel
  • Fonctionnalités attendues :
    • Envoi et affichage de messages en temps réel
    • Affichage de l’heure d’envoi de chaque message
    • Photos de profil pour chaque utilisateur
    • Censure automatique de mots interdits (ex : insultes remplacées par ****)
    • Interface claire et intuitive
  • Cible : Utilisateurs grand public / étudiants
