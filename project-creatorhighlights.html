Creator Highlights

Demande du client :

Le client souhaite un site web vitrine mettant en avant plusieurs créateurs de contenu à travers des cartes présentant leur photo de profil, leur pseudo et leur nombre d’abonnés. Le site doit être clair, moderne et responsive

  • Client : Une plateforme fictive ou une startup spécialisée dans la mise en avant de créateurs digitaux.
  • Objectif : Développer une interface web fonctionnelle et esthétique pour valoriser des profils de créateurs.
  • Contraintes :
    • Codage en HTML et CSS uniquement
    • Mise en page en cartes
    • Design responsive
    • Respect des bonnes pratiques d’intégration web
  • Cible : Utilisateurs de plateformes sociales, recruteurs, marques ou fans de créateurs de contenu.
  • Formats : Site web (HTML / CSS)
